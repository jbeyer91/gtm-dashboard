Add two environment variables in Render to connect your GA4 property:

GA4_PROPERTY_ID Numeric property ID — Admin → Property Settings → Property ID
GA4_SERVICE_ACCOUNT_JSON Full contents of a service account JSON key with Viewer access on the GA4 property

To generate the service account key:

  1. Google Cloud Console → IAM & Admin → Service Accounts → Create
  2. Keys tab → Add Key → JSON → download the file
  3. In GA4: Admin → Property → Property access management → add the service account email as Viewer
  4. Paste the entire JSON file contents into the GA4_SERVICE_ACCOUNT_JSON env var
